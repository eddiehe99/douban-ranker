# douban-ranker

🚀 为你的豆瓣电影和播客自动生成**多榜单排名**。

## 功能

- 聚合 AFI/CC/TSPDT 等电影榜单
- 展示 [中文播客榜](https://xyzrank.com/#/) 排名

## 安装

需要浏览器装有 [Tampermonkey](https://tampermonkey.net/) 或 [Violentmonkey](https://violentmonkey.github.io/) 插件.
